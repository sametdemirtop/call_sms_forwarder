📖 HIZLI BAŞLANGIÇ
═══════════════════════════════════════════════════════════════════════════════

Detaylı talimatlar için: QUICKSTART.md

ADIM 1: Firebase Projesi Oluştur
   → https://console.firebase.google.com/
   → "Proje Ekle" → Proje adı gir → Oluştur

ADIM 2: Firestore Database Aç
   → Firestore Database → Veritabanı Oluştur → Test Mode

ADIM 3: Android & Web Uygulaması Ekle
   → Android: google-services.json indir
   → Web: Firebase config kopyala

ADIM 4: Yapılandırma Dosyalarını Güncelle
   → lib/firebase_options.dart
   → web/index.html
   → web/firebase-messaging-sw.js

ADIM 5: Build & Deploy
   → Android: ./build_android.sh
   → Web: ./deploy_web.sh

ADIM 6: Kullan!
   → Android'de: Servisi başlat
   → iPhone'da: Web sitesini ana ekrana ekle
